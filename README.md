# paxynyae / pax

A Minecraft mod built to implement and test the constructed language **paxəɲæ** inside a real game environment.

This project exists as a practical application of the language — not the other way around.

## Overview

paxəɲæ is a constructed language developed independently as a full linguistic system.

This mod brings it into Minecraft by replacing and extending in-game language assets, allowing paxəɲæ to be used across:
- UI text
- item and block names
- system messages
- gameplay terminology

## Purpose

The goal is to stress-test paxəɲæ as a real working language by applying it at scale in a constrained, interactive system.

Minecraft acts as a sandbox for:
- vocabulary expansion under pressure
- consistency across large translation sets
- morphological stability in real usage contexts

## Role of the mod

The mod is an implementation layer for paxəɲæ.

It exists to:
- host the language in a live environment
- force real-world linguistic scaling
- expose gaps, inconsistencies, and growth opportunities in the conlang
